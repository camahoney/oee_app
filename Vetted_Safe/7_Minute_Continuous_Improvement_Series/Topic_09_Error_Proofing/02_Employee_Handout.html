Error Proofing
Poka-Yoke
🛡
The Golden Rule: The best process is one where it is physically impossible to make a mistake.

The 3 Types

1. Contact
Shape & Fit

Like a SIM card. It has a notched corner so it only fits one way.

2. Fixed Value
Counting

The machine counts 4 welds. It won't release the part until it sees 4.

3. Motion-Step
Sequence

You must close the door before the saw starts. It forces the order.

The Hierarchy

PREVENTION (Best)

(Cannot fail)

DETECTION (Okay)

(Beeps/Lights)

WARNING (Bad)

(Signs/Procedures)

Critical Behaviors

ALWAYS (DO)
  • Test the error-proofing daily (e.g. e-stop).
  • Report if a Poka-Yoke is broken.
  • Ask "Can we make this mistake impossible?"
NEVER (DON'T)
  • NO bypassing safety switches.
  • NO relying on "being careful".
  • NO ignoring the alarm.
