Standard Work
The "One Best Way"
📝
The Golden Rule: Where there is no Standard, there can be no Improvement. We must all start from the same baseline.

The 3 Elements

1. Takt Time
The Customer Pace

How fast must we work to satisfy the customer? (e.g., 1 part every 50 seconds).

2. Sequence
The Order

Do step A, then B, then C. Never C, A, B. Consistency is Quality.

3. Standard WIP
The Inventory

The minimum parts needed on hand to keep the machine running perfectly.

The Shadow Board

🛠
A Place for Everything

If a tool is missing, you should know in 1 second.

5S Rule:
Sort, Straighten, Shine, Standardize, Sustain.

Critical Behaviors

ALWAYS (DO)
  • Follow the SOP exactly.
  • Return tools to the shadow board.
  • Suggest improvements to the SOP.
NEVER (DON'T)
  • NO "winging it".
  • NO hiding tools in your locker.
  • NO ignoring the visual signals.
